# Tutorial Pengisian Form Data Keluarga M.Koesbanoe

Tutorial ini akan memandu Anda langkah demi langkah dalam mengisi Form Data Keluarga M.Koesbanoe.

## Langkah 1: Membuka Form

1. Klik pada link form yang dibagikan kepada Anda.
2. Form akan terbuka di browser Anda, menampilkan judul "Form Data Keluarga M.Koesbanoe" di bagian atas.

![Tampilan Awal Form](https://blog.classy.id/upload/gambar_berita/75dc267299716ea87d51178d7bf9c46c_20250407141238.png)

## Langkah 2: Memilih Generasi

1. Pada langkah pertama, Anda akan diminta untuk memilih generasi.
2. Klik dropdown menu "Pilih Generasi" untuk melihat daftar generasi yang tersedia.
3. Pilih generasi yang sesuai dengan posisi Anda dalam silsilah keluarga M.Koesbanoe.
4. Setelah memilih, klik tombol "Lanjut" untuk melanjutkan ke langkah berikutnya.

**Catatan**: Jika Anda tidak menemukan generasi yang sesuai, hubungi administrator untuk bantuan.

![Pilih Generasi](https://blog.classy.id/upload/gambar_berita/75dc267299716ea87d51178d7bf9c46c_20250407141238.png)

## Langkah 3: Memilih Orang Tua

1. Pada langkah kedua, Anda akan diminta untuk memilih orang tua.
2. Klik dropdown menu "Pilih Orang Tua" untuk melihat daftar orang tua yang tersedia sesuai dengan generasi yang Anda pilih.
3. Pilih orang tua yang sesuai dengan hubungan keluarga Anda.
4. Setelah memilih, klik tombol "Lanjut" untuk melanjutkan ke langkah berikutnya.

**Catatan**: Jika Anda tidak menemukan nama orang tua yang sesuai, hubungi administrator untuk bantuan.

![Pilih Orang Tua](https://blog.classy.id/upload/gambar_berita/6ecff8d4b9d0c1c37bf3725c5d998840_20250407141838.png)

## Langkah 4: Mengisi Data Pribadi

1. Pada langkah ketiga, Anda akan diminta untuk mengisi data pribadi Anda.
2. Isi kolom berikut:
   - **Nama Lengkap**: Masukkan nama lengkap Anda (wajib diisi)
   - **Status**: Pilih status "Hidup" atau "Almarhum/ah"
   - **Nama Pasangan**: Masukkan nama pasangan Anda (jika ada, boleh dikosongkan)
   - **Status Pasangan**: Centang kotak "Alm" jika pasangan Anda sudah almarhum/almarhumah
   - **Nomor HP/WA**: Masukkan nomor HP atau WhatsApp Anda (format bebas, akan dikonversi otomatis)
   - **Alamat**: Masukkan alamat tempat tinggal Anda

3. Setelah mengisi semua data, klik tombol "Lanjut" untuk melanjutkan ke langkah terakhir.

**Tip**: Untuk nomor HP, Anda dapat memasukkan dengan atau tanpa awalan "0". Sistem akan otomatis memformat dengan awalan "62".

![Isi Data Pribadi](https://blog.classy.id/upload/gambar_berita/7ce22a0c928cf4b31148eb3d76706032_20250407141838.png)

## Langkah 5: Menambahkan Data Anak

1. Pada langkah terakhir, Anda dapat menambahkan data anak-anak Anda (jika ada).
2. Jika Anda memiliki anak, klik tombol "Tambah Anak" untuk menambahkan kolom input anak.
3. Masukkan nama lengkap anak Anda di kolom yang muncul.
4. Anda dapat menambahkan beberapa anak dengan mengklik tombol "Tambah Anak" lagi.
5. Jika Anda tidak memiliki anak atau belum ingin menambahkan data anak, Anda dapat langsung melanjutkan ke langkah berikutnya tanpa mengisi data anak.

**Catatan**: Data anak bersifat opsional dan dapat dikosongi jika tidak relevan.

![Tambah Data Anak](https://blog.classy.id/upload/gambar_berita/4c289588e844abb628b4e662747a908f_20250407141838.png)

## Langkah 6: Menyimpan Data

1. Setelah Anda selesai mengisi semua data, klik tombol "Simpan Data" untuk mengirimkan data ke sistem.
2. Tunggu beberapa saat hingga proses penyimpanan selesai.
3. Jika berhasil, Anda akan melihat pesan konfirmasi bahwa data berhasil disimpan.
4. Form akan direset dan Anda dapat mengisi form lagi untuk anggota keluarga lain jika diperlukan.

![Simpan Data](https://via.placeholder.com/600x300/f5f5f5/4F46E5?text=Simpan+Data)

## Tips Pengisian Form

1. **Mengisi Nama**: Nama akan otomatis diformat dengan huruf pertama besar di setiap kata.
2. **Nomor HP**: Sistem akan otomatis mengonversi nomor HP dengan format awalan "62", jadi Anda bisa memasukkan dengan atau tanpa awalan "0".
3. **Kolom Wajib**: Nama adalah kolom wajib, sementara kolom lain bersifat opsional.
4. **Navigasi**: Anda dapat kembali ke langkah sebelumnya dengan tombol "Kembali" jika ingin mengubah data.
5. **Data Anak**: Anda dapat menambahkan banyak anak sesuai kebutuhan, atau tidak menambahkan sama sekali.
6. **Menghapus Anak**: Jika Anda salah memasukkan nama anak, klik ikon tong sampah di sebelah kanan untuk menghapusnya.
